Overflow

Basicamente en este html, se va a explicar el Overflow que sería es desbordamiento.

Tiene 4 valores x defecto

  1. Auto

    2. Optimiza para que se pongan las barras para ir moviendo el visor sobre el contenido, dependiendo de si es necesario, por eje x e y

  2. Hidden

    3. lo que sobresale se oculta y no se puede acceder a esta info, salvo con herramientas externas

  3. Scroll

    4. Se crea una barra de scroll para que se pueda seguir viendo el contenido. En este caso te muestra las 2 barras la de la x y de la y

  4. Visible

    5. No pone ninguna restriccion y muestra todo
