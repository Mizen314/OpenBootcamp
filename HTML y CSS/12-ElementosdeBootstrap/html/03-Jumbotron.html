Custom Jumbotron

Using a series of utilities, you can create this jumbotron, just like the one in previous versions of Bootstrap. Check out the examples below for how you can remix and restyle it to your liking.

Jumbotron

El jumbotron es un texto grande que tiene una llamada a la accion

Explicaciones:

  • bg-light: es background color
  • p: es padding, aca le pone 5
  • py-5: padding top
  • rounded-3, es para darle el efecto redondeado al div
  • fw-bold: es font weight bold
  • display-5: es el tamaño de las letras, como si fueran h1,2,3,4,5 ..
  • btn: button
  • btn-primary: esta la vimos y es una clase para especifica de botones para darle color azul y la forma
  • btn-lg: tamaño del boton
  • fs: font size tambien se maneja como los h...

Trucaso: Si la pagina esta usando bootstrap, esta usando las clases que vienen para trabajar, y como estas son públicas, se pueden copiar, entonces, explorando elemento x elemento, se puede copiar facilmente la estructura, o almenos un aproximado
