Tablas

Lo bueno que tiene esto es que, como todo en bootstrap, es que las propiedades y estilos se agregan con ponerle una clase al elemento. Las mismas pueden ser acumulativas y asi tomar los diferentes estilos. En caso que necesite formatearse màs, se hace un css, para que modifique especificamente lo que no esta bueno. Ver en la documentacion de bootstrap.

  • table sm, hace la tabla màs chica.
  • table strip = le da la intermitencia de colores
  • table-bordered: le pone bordes
  • table-hover: le da la propiedad de hover a la row que se esta seleccionando
