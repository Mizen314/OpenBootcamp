En este caso las columnas funcionan como stack to horizontal, es decir que mientas esten en el tamaño asignado, se van a seguir manteniendo como columnas, pero si estan en un tamaño menor se hacen como filas de "100% width" En este caso para ver todo con las tres columnas, tiene que estar en más de 1486 px. Si se reduce, esto empiezan a alinearse de como filas en vez de columnas. Por defecto siempre se habla de columnas tipo XS.
